# Hacking Lab: Reconnaissance and Web Security

## Overview
This project demonstrates practical cybersecurity skills including reconnaissance, OSINT, and exploitation of web vulnerabilities in a controlled lab environment.

## Objectives
- Perform information gathering and reconnaissance
- Analyze network services and open ports
- Identify and exploit web vulnerabilities
- Understand real-world attack techniques

## Tools Used
- Kali Linux
- Nmap
- Burp Suite
- Shodan
- WHOIS / DNS tools

## Practical Work

### Reconnaissance
- Performed network scanning using Nmap
- Identified active hosts, open ports, and running services
- Analyzed network structure and potential attack surface

### OSINT & Information Gathering
- Collected public data using WHOIS, DNS queries, and Shodan
- Identified domain-related information and exposed services

### Web Security Testing
- Exploited Local File Inclusion (LFI)
- Performed Server-Side Request Forgery (SSRF)
- Demonstrated Remote Code Execution (RCE)

## Key Learnings
- How attackers gather information before exploitation
- Importance of exposed services and misconfigurations
- How web vulnerabilities can lead to full system compromise

## Skills Gained
- Reconnaissance and OSINT techniques
- Vulnerability analysis
- Basic penetration testing
- Use of cybersecurity tools in real scenarios

## Conclusion
This project provided hands-on experience in cybersecurity and demonstrated how multiple vulnerabilities can be combined to compromise a system.
